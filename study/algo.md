Algorithm: LLM-Guided Coverless Image Steganography Using Semantic Hash Mapping (LG-CISH)

1. Overview
The proposed algorithm introduces a coverless steganographic communication framework in which the secret message is not embedded by modifying pixel values. Instead, a Large Language Model (LLM) semantically encodes the message and maps it to robust perceptual hashes of images stored in a predefined image database.
Each semantic token generated by the LLM corresponds to a unique image hash, and the sequence of retrieved images forms the stego representation of the secret message.
Thus, the system ensures:
Zero pixel modification
High resistance to steganalysis
Semantic robustness

7. Security Properties
The algorithm provides:
1. Steganalysis Resistance
Since no pixel values are modified, classical detectors cannot identify hidden data.

LLM-generated tokens make reverse decoding difficult.
3. Hash Collision Protection
Perceptual hashing ensures robustness against minor image transformations.
